# kennymatsudo/homebrew-tap

Personal Homebrew tap for kennymatsudo's projects.

## Install

```
brew install --cask kennymatsudo/tap/kpm
```

Or tap once and install by name:

```
brew tap kennymatsudo/tap
brew install --cask kpm
```

The app is ad-hoc signed rather than notarized, so the cask clears the macOS quarantine flag on install and Gatekeeper opens it normally. Update later with `brew upgrade`.

`brew help`, `man brew`, or [Homebrew's documentation](https://docs.brew.sh).
